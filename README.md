# QuickChat

## Live Demo:  https://chat-app-pi-sage.vercel.app



## Overview

QuickChat is a real-time full-stack chat application built using the MERN stack and Socket.IO. Users can create accounts, update profiles, and exchange messages instantly through a responsive web interface.

## Features

- Real-time messaging using Socket.IO
- User authentication and authorization
- Profile picture support
- Online/offline user status
- Responsive UI
- Cloud database integration
- Secure API communication
- Modern chat interface

## Technologies Used

- MongoDB
- Express.js
- React.js
- Node.js
- Socket.IO
- JWT Authentication
- Vercel Deployment
